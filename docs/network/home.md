# Home Networks

For security reasons I run 3 separate networks after my fiber router. A TP-Link TL-SG105E v5 separates the following two Subnets. Two ports are reserved for a future OPNSense firewall.

## Lan for guests

The Hofkoh network exist since 2013 and reminds me how fast time changes. Being just 2.4 GHz it was limited in speed until I upgraded to Wifi 6 in 2026.

## IOT for iot devices

It started small, but now there are two Yeelight lightbulbs, two Tuya smartswitches and 2 Feinstaubsensor AQI monitors with an ESP8266 that run 24/7
